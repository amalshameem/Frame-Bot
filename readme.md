# Frame Bot

## Introduction

A simple discord bot that puts a frame on images 

## Requirements

- Access to Discord Developer Portal, Bot token
- Python 3.14+

## Installation

- Create an additional file called config.py and put your bot token there
- Install all the requirements in requirements.txt (note: all the modules in it might not be needed)
- Run main.py

## Usage of AI and Image processing

The bot is built with a autoframing mechanism which uses yolo model for detection of various subject.
